##### M5-2：聊天面板 `[ ]`

- **目标**：流式消息渲染 + 工具调用状态显示。
- **文件**：`src/ui/chat_panel.py`
- **实现**：

- **验收**：输入消息 → 流式逐字输出 → 工具调用时显示状态
