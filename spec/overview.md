## 1. 项目概述

stu-agent 是一款面向中国大学生的桌面 Agent 开发项目。采用模板驱动的结构化文档编写助手作为核心场景，通过渐进式里程碑交付——从命令行对话到完整桌面应用。构建过程中自研 Agent Core、工具系统和 Skill 框架。

核心理念：课程论文和实验报告本质是同一件事——「按模板结构、逐章填充、最终导出」。统一为模板驱动后，新增文档类型只需新增模板，无需改动 Agent 逻辑。

### 设计理念

本项目既是实用工具，也是**学习与实战项目**：
- **教是最好的学**：通过自研 Agent Core，深入理解 LLM Agent 的底层原理
- **简约但不简陋**：参考 pi-core 的架构风格——事件驱动、工厂模式工具、每工具一文件
- **开箱即用**：PyInstaller 打包为独立安装包，学生双击即可使用

### 核心交互流程

```
学生选择模板（课程论文 / 实验报告 / ...）
    ↓
学生输入主题
    ↓
Agent 生成大纲 → 学生确认/修改
    ↓
逐章节写作 → 学生逐章审查
    ↓
全文润色 → 学生最终确认
    ↓
一键导出 Word (.docx)
```

## 2. 核心功能

### 2.1 核心能力

| 能力 | 描述 | 里程碑 |
|------|------|--------|
| **流式对话** | 多轮对话，逐字输出，LLM 可插拔（DeepSeek/Qwen/OpenAI） | M1 |
| **工具调用** | LLM 调用基础工具（read_file / write_file / bash），自动执行并喂回 | M2 |
| **Skill 系统** | YAML 定义业务能力（system_prompt + 工具列表），与基础工具解耦 | M3 |
| **模板驱动写作** | 写作 Skill：读模板 → 大纲 → 逐章写 → 导出 Markdown | M4 |
| **桌面 GUI** | NiceGUI 前端：聊天面板 + 文档预览 + 设置，WebSocket 实时 | M5 |
| **打包发布** | PyInstaller 单文件可执行，开箱即用 | M6 |

### 2.2 内置模板

课程论文模板：`摘要 → 关键词 → 引言 → 文献综述 → 正文 → 结论 → 参考文献`

实验报告模板：`实验目的 → 实验原理 → 实验器材 → 实验步骤 → 实验数据 → 结果分析 → 结论`

> 扩展性：新增模板只需在 `config/templates/` 添加 YAML，不改代码。

### 2.3 用户故事

1. **M1-M2**：用户在终端输入 `python main.py`，与 Agent 对话，Agent 可读文件、写文件、执行命令。
2. **M4**：用户说"写实验报告"→ Agent 读模板 → 生成大纲 → 逐章写 → 导出 `.md` 文件。
3. **M5**：浏览器打开 → 三栏布局 → 流式对话 → 文档实时预览 → 一键导出。
